# Config Server - Microservices Lab ⚙️

Este repositório centraliza todas as configurações externas (`.properties`) utilizadas no ecossistema de microsserviços.

## 🔗 Repositório Principal
O código-fonte dos serviços que consomem estas configurações pode ser encontrado em:
👉 [spring-cloud-microservices-lab](https://github.com/1LucasSena/spring-cloud-microservices-lab)

## 📁 Estrutura de Arquivos
* `service-one.properties`: Configurações de teste, mensagens e conectividade do Service One.
